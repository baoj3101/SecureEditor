Instructions to use Editors

There are four types of editors available: Text Editor, Rich Editor, HTML Editor, and Secure Editor. This document provides instructions on how to use each type of editor.

Text Editor

This is similar to Windows NotePad. It's a simple plain text editor that support copy, cut, and paste.

  • Copy: Ctrl+C
  • Cut: Ctrl+X
  • Paste: Ctrl+V

To save to a file, use File menu "Save File". To load from a file, use File menu "Open File". To exit, use File Menu "Exit". To open this help message, use Help Menu "How To"

Rich Editor

This is similar to Windows Rich Editor: one can customize font family, font size, font style, and color.

  • Font Family: All fonts installed in system are listed. Use Font Family ComboBox to choose.
  • Font Size: Use Font Size ComboBox to choose.
  • Font Style: Use Font Style buttons to choose Bold (B), Italic (I), and Underline (U)
  • Font Color: Click on Color button to open color chooser.

File Open, Save, and Exit behaves the same as the Text Editor.

HTML Editor

This is a text editor that provide HTML rendering. One can use text edit area to edit HTML file.

  • To render in HTML, use Tool Menu "Show HTML"
  • To switch back to plain text mode, use Tool Menu "Show Plain"

File Open, Save, and Exit behaves the same as the Text Editor.

Secure Editor

This is a text editor that can save content to file as encrypted. When content is saved to a file, the content is first encrypted and then encoded. When a file is loaded, the content will be decoded followed by decryption.

  • To show content as encrypted, use Tool Menu "Show Encrypted"
  • To show content as clear text, use Tool Menu "Show Decrypted"

File Open, Save, and Exit behaves the same as the Text Editor.
